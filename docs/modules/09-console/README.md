# 模块 09：管理控制台

## 完成范围

- 新增类型化 API Client，统一处理 API Base、错误和 JSON 请求。
- 对话工作台支持会话 ID、查询结果和安全审批提案。
- 审批中心读取审批列表并展示状态。
- 知识库页面支持 Markdown/TXT 上传并显示入库片段数量。
- 运行轨迹页面读取审计事件，避免展示隐藏思维链。
- 评估中心提供 Recall、工具正确率和安全准确率展示位。
- 使用深色运营控制室视觉系统、响应式布局、键盘焦点和 reduced-motion 兼容。

## 架构边界

- 页面通过 `web/src/api/client.ts` 访问后端，不直接拼接业务请求。
- 后续拆分 Vue Router/Pinia 页面时保持 API Client 不变。
- 前端不保存后端密钥，认证 Token 由后续生产登录流程注入。

## 验证结果

- `npm run build`：通过。
- 页面构建产物生成于 `web/dist`。
